# Example Dental Clinic — AI Chatbot Demo

A demo dental practice website with an embedded AI assistant that answers patient questions, verifies insurance coverage, triages emergencies, and captures appointment requests 24/7.

Built as a portfolio piece to demonstrate what an AI receptionist looks like in context on a real dental practice site.

## What It Does

The AI assistant on this demo site handles the inquiries that normally slip through the cracks of a busy dental practice:

- **24/7 availability** — answers patient questions at 9pm, 2am, or any time the front desk is closed
- **Insurance verification** — instantly confirms whether the practice accepts specific providers (Blue Cross, Manulife, Sun Life, CDCP, and more)
- **Emergency triage** — assesses severity of dental emergencies and routes patients appropriately
- **Appointment intake** — captures name, phone number, preferred time, and reason for visit so the office can follow up
- **Service information** — answers questions about cleanings, Invisalign, pediatric care, restorative work, and more
- **Lead capture** — no patient inquiry goes unanswered, even outside business hours

## Why This Matters

The average dental practice loses 5-8 new patients a month to unanswered after-hours inquiries. At roughly $2,000 in lifetime value per patient, that's $10,000-$15,000 of lost revenue every month.

This system recovers that revenue by making sure every patient question gets a fast, specific, helpful answer — and every booking request gets captured as a lead.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript (parallax template by [TemplateMo](https://templatemo.com/tm-612-parallax-starter))
- **Chatbot:** [Voiceflow](https://www.voiceflow.com/) AI agent with custom dental-specific instructions
- **Hosting:** Static site, deployable to GitHub Pages, Netlify, Vercel, or any web host

## Running Locally

1. Clone this repository
2. Open the folder in a code editor (VS Code recommended)
3. Use the Live Server extension (or any local web server) to serve `index.html`
4. The chatbot widget loads automatically in the bottom-right corner
